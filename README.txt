A billing software is designed to handle time, billing tracking and invoicing customers for services and products. 
This software can -
1. Calculate Total Price
2. Calculate Tax
3. Generate Bill
4. Save Bill
5. Search Bills
6. Clear Data
7. Exit


To run this project - 
1. In folder 'billing_software'. There is a single file(bill.py) and one folder(bills).
2. Run 'bill.py'  file in python editor.
3. Folder 'bills' contain all saved bills(in .txt format).
